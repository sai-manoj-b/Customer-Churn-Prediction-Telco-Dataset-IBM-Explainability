# Customer-Churn-Prediction-Telco-Dataset-IBM-Explainability (ML + SHAP)
Built an end-to-end churn prediction pipeline (EDA, feature engineering, ensemble models) on 7K+ customer records, achieving high accuracy. 
Applied SHAP explainability and feature importance to identify key churn drivers (tenure, contract type, monthly charges).

---

## Objectives
- Analyze customer behavior and churn patterns
- Build and evaluate machine learning models
- Explain model predictions using SHAP
- Derive actionable business insights


---

## Key Insights
- Contract type, tenure, and pricing-related features are strong churn drivers
- Tree-based models outperform baseline logistic regression
- SHAP confirms feature impacts and improves business interpretability

---

## Business Value
- Enables targeted retention strategies
- Provides explainable ML outputs for stakeholders
- Bridges the gap between analytics and decision-making
